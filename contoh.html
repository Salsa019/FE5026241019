Section 01 / 09 Basic Tables
Basic table Bootstrap 4 memiliki light padding (jarak dalam yang tipis) dan horizontal dividers (garis pemisah antar baris). Class .table menambahkan styling dasar pada tabel. Gunakan <thead> untuk kepala tabel, <th> untuk sel header, <tbody> untuk isi, <tr> untuk baris, dan <td> untuk sel data. 
<table class="table">
  <thead>
    <tr>
      <th>Nama</th>
      <th>NRP</th>
      <th>Kelas</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Salsa</td>
      <td>5026241019</td>
      <td>B</td>
    </tr>
    <tr>
      <td>Aludra</td>
      <td>5026241098</td>
      <td>B</td>
    </tr>
    <tr>
      <td>Naufatuzaki</td>
      <td>5026241032</td>
      <td>D</td>
    </tr>
  </tbody>
</table>

Section 05 / 09 Black/Dark Table
Class .table-dark menambahkan latar belakang hitam pada tabel. Dapat dikombinasikan dengan .table-striped untuk baris selang-seling gelap, .table-hover untuk efek sorot, atau .table-borderless untuk tampilan tanpa garis sama sekali. 
<!-- Dark table -->
<table class="table table-dark"> ... </table>

<!-- Dark + striped -->
<table class="table table-dark table-striped"> ... </table>

<!-- Dark + hover -->
<table class="table table-dark table-hover"> ... </table>

<!-- Borderless -->
<table class="table table-borderless"> ... </table>

Section 06 / 09 Contextual Classes
Kelas kontekstual digunakan untuk memberikan makna visual pada data. Dapat diterapkan pada <table>, <tr>, atau <td>: .table-primary (biru), .table-success (hijau), .table-danger (merah), .table-warning (oranye), .table-info (biru muda), .table-secondary (abu-abu), .table-light, dan .table-dark. 
<table class="table">
  <tbody>
    <tr class="table-primary"> ... </tr>
    <tr class="table-success"> ... </tr>
    <tr class="table-danger">  ... </tr>
    <tr class="table-warning"> ... </tr>
    <tr class="table-info">    ... </tr>
  </tbody>
</table>

Section 07 / 09 Table Head Colors
Class .thead-dark menjadikan kepala tabel berwarna hitam dengan teks putih. Class .thead-light menjadikan kepala tabel berwarna abu-abu muda dengan teks gelap. Selain itu, class .table-sm dapat digunakan untuk menghemat ruang dengan memotong padding sel menjadi setengahnya. 
<!-- Dark head -->
<table class="table">
  <thead class="thead-dark"> ... </thead>
</table>

<!-- Light head -->
<table class="table">
  <thead class="thead-light"> ... </thead>
</table>

<!-- Small table -->
<table class="table table-sm"> ... </table>

Section 08 / 09 Responsive Tables
Membungkus tabel dengan .table-responsive menambahkan scrollbar horizontal jika konten terlalu lebar. Tersedia varian breakpoint: .table-responsive-sm (aktif di <576px), .table-responsive-md (aktif di <768px), .table-responsive-lg (aktif di <992px). 
<!-- Selalu responsive -->
<div class="table-responsive">
  <table class="table"> ... </table>
</div>

<!-- Responsive hanya di layar kecil (<576px) -->
<div class="table-responsive-sm">
  <table class="table"> ... </table>
</div>

Section 09 / 09 Study Case
Penerapan seluruh fitur tabel Bootstrap 4 secara bersamaan dalam satu tabel Daftar Nilai Mahasiswa. Tabel ini menggabungkan: .table-bordered untuk garis tepi, .table-hover untuk efek sorot, .table-sm untuk padding ringkas, .thead-dark untuk kepala tabel gelap, kelas kontekstual (.table-success, .table-warning, .table-danger) untuk menandai status nilai, serta menggunakan .table-responsive agar responsive di layar kecil (dapat digeser). 
<div class="table-responsive">
  <table class="table table-bordered table-hover table-sm">
    <thead class="thead-dark">
      <tr>
        <th>No</th>
        <th>Nama</th>
        <th>NRP</th>
        <th>Kelas</th>
        <th>Nilai</th>
        <th>Grade</th>
        <th>Status</th>
      </tr>
    </thead>
    <tbody>
      <!-- Lulus -->
      <tr class="table-success"> ... </tr>
      <!-- Perlu perbaikan -->
      <tr class="table-warning"> ... </tr>
      <!-- Tidak lulus -->
      <tr class="table-danger">  ... </tr>
    </tbody>
  </table>
</div>
